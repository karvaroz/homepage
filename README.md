# My Personal Website

Welcome to my personal website repository! This site serves as an online résumé where I showcase my skills, projects, and experiences. Feel free to explore and get to know more about me.

## Overview

This repository contains the source code and assets for my personal website. The site is designed to provide a glimpse into my professional background, skills, and the projects I've worked on. Whether you're a potential employer, colleague, or just curious, I hope you find the information here helpful.

## Features

- **Résumé:** A detailed overview of my professional experience, education, and skills.
- **Projects:** Highlights of some of the key projects I've worked on, complete with descriptions and links.
- **Contact:** Ways to get in touch with me. I welcome any inquiries, feedback, or collaboration opportunities.

## Technologies Used

- HTML
- CSS
- JavaScript
- Node Js / Express Js
- React Js
- ASP.NET Core

## How to Use

1. Clone the repository: `git clone https://github.com/karvaroz/homepage.git`
2. Open the `index.html` file in your web browser.

Feel free to fork this repository and customize it for your own use!
